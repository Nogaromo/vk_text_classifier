# Описание файлов
* В папке data лежат файлы трейна и теста
* В папке utils лежат 2 файла (get_embeddings.py, text_preprocessing.py)
  * get_embeddings.py - с помощью него можно получить эмбеддинги для текстов
  * text_preprocessing.py - делает небольшую предобработку текста (спойлер: она особо ситуацию лучше не делала, поэтому я ее убрал)
* bert_fine_tune.ipynb - здесь просто дообучение берта для классификации
* fast_text.ipynb - применение модели fasttext для классификации
* lstm.ipynb - самописная сеть для классификации
* pure_catboost.ipynb - использовнаие `text_features` для `catboost`
* training_on_embeddings.ipynb - обучение кэтбуста и полносвязной сетки на эмбеддингах
* test_preds.csv - файл со скорами для теста
# Лучшая модель
В качестве итоговой модели для предсказаний я выбрал полносвязную сетку, обученную на эмбеддингах с берта, тк это давало лучший результат.
